# 远程桌面隐私政策

生效日期：2026-06-30

本隐私政策适用于 HarmonyOS NEXT 应用”RemoteDesktop”。本应用用于在你的设备和你配置的远程主机之间建立 RDP、RustDesk、SSH 连接，并预留 VNC 等扩展入口；同时提供主机配置管理、华为账号同步、密钥保险库、TOTP 验证器和本地安全保护能力。

## 一、我们收集和处理的信息

本应用仅在实现功能所必需的范围内处理数据：

1. 账号信息：华为账号 UnionID、昵称、头像，用于账号登录和跨设备同步。
2. 主机配置：主机名称、IP 或域名、端口、协议类型、分组、显示配置、RustDesk 中继配置。
3. 敏感凭据：远程主机密码、SSH 私钥、TOTP secret、RustDesk 中继账户密码。
4. 应用偏好：主题、强调色、壁纸、终端字号、RDP/RustDesk 画质与控制模式等设置。
5. 权限相关数据：相机实时画面仅用于扫描 TOTP 二维码，不保存照片或视频；生物识别由系统处理，本应用只接收认证通过或失败结果。

本应用不收集远程桌面屏幕内容、SSH 会话历史、精确位置、通讯录、短信、通话记录、已安装应用列表、IMEI 或序列号。

## 二、数据用途

1. 建立远程连接：根据你保存的主机配置发起 RDP、RustDesk 或 SSH 连接；VNC 等扩展入口以后续版本实际能力为准。
2. 云同步：在你登录华为账号后，将主机、密钥、TOTP、RustDesk 中继等数据同步到华为 Cloud DB。
3. 安全保护：使用主密码、AES-256-GCM 加密、生物识别验证和 TOTP 隐私保护，降低敏感数据被未授权访问的风险。
4. 个性化体验：根据你的主题、壁纸、窗口尺寸和控制模式偏好调整界面。

## 三、权限说明

1. `ohos.permission.INTERNET`：用于远程连接、云同步和 RustDesk 中继通信。
2. `ohos.permission.CAMERA`：仅在你主动扫描 TOTP 二维码时申请，用于识别二维码内容。
3. `ohos.permission.ACCESS_BIOMETRIC`：仅在你启用主机安全锁或 TOTP 隐私保护时用于系统认证。
4. `ohos.permission.DETECT_GESTURE`：用于握持手势检测和界面适配优化。

拒绝相机或生物识别权限不会影响其他基础功能。

## 四、数据存储与保护

敏感数据使用主密码派生的数据加密密钥进行 AES-256-GCM 加密。主密码不会明文存储，数据加密密钥仅存在于内存中。你忘记主密码后，我们无法帮你恢复已加密数据。

云同步数据通过华为 Cloud DB 传输和保存。敏感字段在离开设备前已加密。远程桌面视频、音频、输入和会话内文本数据在连接过程中传输，不由本应用服务器中转或保存。

## 五、第三方共享

本应用不集成广告 SDK，不进行用户画像，不向第三方出售或共享你的个人信息。

本应用使用华为官方 Kit 提供账号、云同步、生物识别、加密和关键资产存储能力。相关数据处理遵循华为服务的规则。RustDesk 中继服务器由你自行选择或配置，本应用不是中继服务提供方。

## 六、你的权利

你可以在应用内查看、修改、删除主机、密钥、TOTP 和中继配置。你可以退出华为账号以停止云同步，也可以在设置中关闭或重置加密。重置加密会清空相关数据，请谨慎操作。

你也可以通过系统设置撤回相机、生物识别等权限授权。

## 七、未成年人保护

本应用不面向未成年人设计。如未成年人需要使用，应在监护人陪同和同意下使用。

## 八、政策更新

隐私政策发生重大变化时，我们会在应用内或发布说明中提示。继续使用本应用即表示你已了解更新后的政策。
