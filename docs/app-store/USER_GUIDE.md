# RemoteDesktop 用户使用文档

更新日期：2026-07-14

## 一、应用简介

RemoteDesktop 是面向 HarmonyOS NEXT 的原生远程连接工具。1.0.6 当前可使用 RDP、RustDesk 和 SSH/SFTP，并提供密钥库、TOTP 验证器、华为云同步、本地备份、后台实况以及手机、平板和 PC 响应式界面。VNC 原生适配代码仍在验证中，当前版本不会把它展示为可用连接能力。

更新安装后，1.0.6 的主要变化只会在第一次进入应用时自动展示一次；手机使用全屏更新说明，平板和 PC 使用半屏弹窗。新安装应用会显示完整的新用户功能介绍。

## 二、首次使用

1. 阅读首次启动功能介绍，了解连接、输入、传输、同步和安全能力。
2. 选择华为账号登录或离线使用。离线模式的数据只保存在当前设备。
3. 如需跨设备同步，请登录华为账号，并按设置页提示完成云服务前置条件。
4. 如需保护密钥、凭据和验证器数据，请设置并妥善保存主密码。主密码丢失后无法恢复已加密数据。
5. 教程可从“设置 → 教程 → 简单使用教程”随时重新打开。

## 三、添加和管理主机

在主机页点击添加按钮，按场景选择连接方式：

1. RDP：适合连接已开启远程桌面的 Windows 专业版或企业版设备。填写主机地址、端口、Windows 凭据和显示配置。
2. RustDesk：适合跨平台远程控制。填写远程设备 ID 和密码；使用自建服务时，先配置 ID、Relay 和密钥信息。
3. SSH：适合服务器终端维护，支持密码、私钥和跳板机。首次连接前请核对服务器主机指纹。
4. SFTP：建立 SSH 会话后，从顶部文件入口浏览和管理远端文件。

主机可按工作组整理并按协议筛选。界面展示的剪贴板、文件传输等能力以当前真实会话状态为准；尚未就绪时会说明原因。

## 四、远程控制与输入

RDP 和 RustDesk 支持触控板、直接触控与键鼠控制。手机和平板可使用触控手势，PC 更适合物理键盘鼠标；三指触摸远程画面可唤起控制面板和系统键盘。

系统输入法支持中文、Emoji 和大段文字，并按输入顺序发送到远端。远程输入仍受目标系统、应用和当前协议能力影响。

## 五、文件传输与剪贴板

- SFTP 支持文件浏览、上传、下载、新建目录、重命名和删除。
- 上传或下载开始后，传输状态区域会提供取消操作。
- 保留在远端的部分上传文件可按已确认位置安全续传。
- RDP 通过共享目录提供文件访问；RustDesk 按当前会话能力提供文件传输。
- RDP 与 RustDesk 支持双向纯文本剪贴板同步，并包含长度限制和回环保护。

文件能力只有在远端协议确认后才会显示为可用，连接成功并不代表所有可选能力都已经就绪。

## 六、密钥库、2FA 与隐私保护

密钥库用于保存 SSH 私钥，2FA 验证器用于保存 TOTP 一次性验证码。添加 2FA 时可扫描二维码，也可手动输入标准 `otpauth` 信息。

设置页可以启用验证器隐私保护和主机安全锁。敏感数据使用本地主密码派生密钥进行 AES-256-GCM 加密；请勿把主密码、私钥、验证码种子或备份文件发送给他人。

## 七、云同步、离线与本地备份

登录华为账号后，可以选择需要同步的数据类型。云同步会检查必要前置条件，并提供失败重试、下载回滚和异常恢复；敏感字段在上传前先在本地加密。

离线模式不会把业务数据上传到云端。本地备份支持导出数据、导入前预检和事务恢复：无效备份不会覆盖现有数据，恢复失败时会回滚。备份可能包含敏感信息，应存放在可信位置。

## 八、后台实况与多设备界面

远程会话支持后台实况和返回应用后的画面恢复。手机使用适合触控的布局，平板和 PC 使用更宽的响应式布局；远程会话与主机页保留沉浸式光感悬浮底栏。

后台能力可能受系统电量策略、窗口状态和设备权限影响。长时间离开后如远端已断开，请按界面提示重新连接。

## 九、反馈与畅联群聊

打开“设置 → 反馈”，可在两个入口间切换：

1. 邮箱反馈：用于提交问题、建议和必要的非敏感复现信息。
2. 加入畅联群聊：在线获取当前有效的群聊二维码，加载失败时可重试，加载成功后可保存到系统相册。

群聊二维码通过 HTTPS 清单更新并设有有效期。请勿在邮件或群聊中发送密码、主密码、私钥、TOTP 种子、真实主机地址或完整数据库备份。

## 十、常见问题

1. 无法登录华为账号：确认当前应用签名、包名和 AppGallery Connect 配置匹配，并检查网络和账号状态。
2. RustDesk 连接失败：检查 ID 服务器、Relay 服务器、公钥、远程设备 ID 和密码。
3. RDP 登录失败：确认 Windows 已开启远程桌面，系统版本支持作为服务端，并核对本地、域、Microsoft 或 Azure AD 账号格式。
4. SFTP 看不到取消按钮：确认当前确实存在进行中的上传或下载，并查看文件面板的传输状态区域。
5. 剪贴板没有同步：确认设置中启用了对应协议的文本剪贴板，并等待会话能力显示为就绪；当前只同步纯文本。
6. 忘记主密码：主密码无法找回，只能重置加密并清空无法解密的相关数据。
