# 远程桌面用户使用文档

更新日期：2026-06-30

## 一、应用简介

远程桌面是面向 HarmonyOS NEXT 的原生远程连接工具，提供 RDP、RustDesk、SSH 连接能力，并预留 VNC 等扩展入口。当前 RustDesk、SSH、密钥保险库、TOTP 验证器、云同步、PC 多端适配和沉浸式设置体验已经完成阶段性开发；RDP 真机完整链路、VNC 真实接入、主机安全锁持久化和发布测试仍需继续验证。

## 二、首次使用

1. 打开应用后阅读引导页。
2. 选择华为账号登录或离线模式。
3. 如需云同步，请登录华为账号并设置主密码。
4. 如只在本机使用，可以跳过登录，数据仅保存在本地。

## 三、添加主机

在主机页点击添加按钮，选择协议并填写连接信息：

1. RDP：填写 Windows 主机地址、端口、Windows 凭据和显示配置。
2. RustDesk：先在中继页配置 ID/Relay 服务器，再填写远程设备 ID 和密码模式。
3. SSH：填写服务器地址、端口、用户名，可选择密码、私钥或跳板机。
4. VNC：当前为预留入口，后续以实际版本能力为准。

## 四、连接与控制

RustDesk 和 RDP 连接页支持触控板、直接触控、键鼠三类控制模式。PC 大屏默认更偏向物理键盘鼠标；手机和平板默认更偏向触控板操作。SSH 终端支持软键盘、物理键盘、文本粘贴、方向键和常用功能键。系统级双向剪贴板、麦克风转发和 VNC 完整连接能力仍以后续版本实际实现为准。

## 五、密钥保险库与 2FA

密钥保险库用于保存 SSH 私钥和 TOTP 一次性验证码。添加 2FA 时可扫描二维码，也可手动输入 otpauth 信息。设置页可启用 TOTP 隐私保护，启用后查看验证码前需要系统认证。

## 六、云同步与加密

登录华为账号后，主机、密钥、TOTP 和 RustDesk 中继配置可以跨设备同步。敏感字段会先在本地加密，再同步到云端。主密码丢失后无法恢复加密数据，这是安全设计的一部分。

设置页支持启用、解锁、关闭和重置加密。重置加密会清空相关数据，操作前请确认已经备份必要信息。

## 七、常见问题

1. 无法登录华为账号：确认 AppGallery Connect 已开通 Account Kit，并且当前包名、签名和 AGC 配置一致。
2. RustDesk 连接失败：检查 ID 服务器、中继服务器、公钥、远程设备 ID 和密码。
3. RDP 登录失败：确认 Windows 已开启远程桌面，账号格式正确，Microsoft 账号场景可尝试使用 Windows 本机映射账号。
4. TOTP 扫码失败：确认已授予相机权限，二维码为标准 otpauth URI。
5. 忘记主密码：无法恢复原加密数据，只能重置加密并清空相关数据。
