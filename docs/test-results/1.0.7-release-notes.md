# 1.0.7 Release Notes Startup Matrix

| Scenario | Expected startup mode |
| --- | --- |
| Fresh install (`guide_shown_v4=false`) | Existing first-install tutorial |
| Existing install without a release marker | 1.0.7 release notes |
| Upgrade from marker `1.0.6` | 1.0.7 release notes |
| Restart after completing 1.0.7 notes | Login page |
| Force-close before completing notes | 1.0.7 release notes again |
| Stored marker newer than current version | Login page |
| Malformed stored marker | Login page |

Release-note completion writes only `last_seen_release_version=1.0.7`. First-install
tutorial completion writes both `guide_shown_v4=true` and the current release marker.

The 1.0.7 Swiper covers only the changes included between the 1.0.6 baseline and
this version: remote IME session stability, RDP certificate trust management, and
RustDesk stream/reconnect lifecycle hardening.
