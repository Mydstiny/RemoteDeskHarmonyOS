# 1.0.8 Release Notes Startup Matrix

| Scenario | Expected startup mode |
| --- | --- |
| Fresh install (`guide_shown_v4=false`) | 1.0.8 first-install tutorial |
| Existing install without a release marker | 1.0.8 release notes |
| Upgrade from marker `1.0.7` | 1.0.8 release notes |
| Restart after completing 1.0.8 notes | Login page |
| Force-close before completing notes | 1.0.8 release notes again |
| Stored marker newer than current version | Login page |
| Malformed stored marker | Login page |

Release-note completion writes only `last_seen_release_version=1.0.8`. First-install
tutorial completion writes both `guide_shown_v4=true` and the current release marker.

The 1.0.8 Swiper covers the user-visible additions and fixes in this release:
background remote video/PIP, AVSession media identity, VP9 software-decoder
shutdown ordering, false PIP prevention, startup grip permission handling, and
responsive onboarding/tutorial sheets.
