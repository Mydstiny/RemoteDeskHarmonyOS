# 云同步手动测试清单

> RemoteDesktop — 端云分布式数据库同步验证

## 前置条件

- [ ] **设备 A**: MatePad Mini (`192.168.31.223:40123`) 已登录华为账号
- [ ] **设备 B**: 手机 (`192.168.31.222:38451`) 已登录**同一**华为账号
- [ ] 两台设备均连接 **WiFi** (云同步策略: WiFi only)
- [ ] AGC 项目已开启 **Cloud DB** 服务 (AppGallery Connect → 我的项目 → RemoteDesktop → Cloud DB)
- [ ] 应用 `AppScope/app.json5` 中 `cloudStructuredDataSyncEnabled: true`
- [ ] 两部设备均已安装最新构建的应用

## 监控日志

在测试过程中，通过以下命令实时查看同步日志:

```bash
# 设备 A
hdc -t 192.168.31.223:40123 shell hilog -T CloudSync

# 设备 B
hdc -t 192.168.31.222:38451 shell hilog -T CloudSync
```

关键日志 marker:
- `CloudStore 初始化完成` → RDB + 分布式表就绪
- `轮询拉取完成` → 15s 定时拉取成功
- `pushTable 完成` → 手动推送成功
- `主机同步完成 ✓` → 主动同步完成
- `HostSyncService.loadFromCloud` → 从云加载数据
- `insertHost/updateHost/deleteHost` → CRUD 操作
