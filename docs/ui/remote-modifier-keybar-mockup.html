远程桌面修饰键 UI

这是接入正式页面前的视觉稿。它遵循中转站里 Claude 留下的规则：移动端不恢复旧底部工具栏，只保留低遮挡的输入工具条；控制入口仍由三指面板或键盘按钮拉出。

短按 Ctrl / Alt / Shift / Win 进入一次性状态。下一个按键发送组合键后自动释放。
长按 修饰键进入锁定状态，右上角显示“锁”。再次点击同一个键释放。
Fn 不是远程真实按键，只切换功能层，展示 F1-F12、Home、End、PgUp、PgDn、Ins、PrtSc。
快捷键 常用组合键收进右侧面板，避免主工具条过宽。Ctrl+Alt+Del 作为协议特殊动作独立显示。
一次性: 蓝色半透明 锁定: 高亮 + 锁标 Fn层: 绿色提示 危险动作: 红色
