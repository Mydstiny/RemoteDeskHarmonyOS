===== 媒体资源占位说明 =====

本目录存放应用所需图标和图片资源。
请使用 DevEco Studio 的 Image Asset 工具生成以下资源：

必要资源（需自行添加）:
- icon.png        — 应用图标 (1024x1024, PNG)
- ic_hosts.png    — 主机列表导航图标 (48x48)
- ic_connection.png — 连接导航图标 (48x48)
- ic_settings.png — 设置导航图标 (48x48)
- ic_search.png   — 搜索图标 (24x24)
- ic_add.png      — 添加按钮图标 (24x24)
- ic_back.png     — 返回图标 (24x24)
- ic_user.png     — 用户头像 (80x80)
- default_host.png — 主机默认图标 (96x96)
- icon_locked.png — 锁定图标 (48x48)
- icon_unlocked.png — 解锁图标 (48x48)
- ic_quick_connect.png — 快速连接图标 (24x24)
- ic_import.png   — 导入图标 (24x24)
- ic_export.png   — 导出图标 (24x24)
- ic_folder.png   — 文件夹图标 (24x24)
- ic_clipboard.png — 剪贴板图标 (24x24)
- ic_printer.png  — 打印机图标 (24x24)
- ic_camera.png   — 摄像头图标 (24x24)
- ic_mic.png      — 麦克风图标 (24x24)
- ic_smartcard.png — 智能卡图标 (24x24)
- ic_keyboard.png — 键盘图标 (24x24)
- ic_gateway.png  — 网关图标 (24x24)
- ic_chevron_down.png — 展开箭头 (24x24)
- ic_chevron_right.png — 折叠箭头 (24x24)

格式要求:
- 应用图标: 1024x1024 PNG
- 导航/功能图标: 48x48 PNG, 白色单色
- 小图标: 24x24 PNG, 白色单色
- 系统会自动适配多密度 (120/160/240/320/480/640 dpi)
